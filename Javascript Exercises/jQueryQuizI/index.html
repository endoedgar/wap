jQuery Quiz I

  1. In jQuery terminology, what is the difference between the jQuery function and the jQuery object?
    The jQuery function (jQuery() or $()) allows you to run jQuery methods on a specified jQuery object, CSS Selector or HTML code. The jQuery Object can be a collection of DOM objects which you can use with the jQuery function to run some jQuery methods.

  2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red.

    Sample HTML:

    <body> <h1>abc</h1><br><div> <h1>div-1</h1> <h1>div-2</h1> </div> <h1>xyz</h1> </body>
    $(document).ready(function(){ $('div h1').css('background-color', 'red'); });
