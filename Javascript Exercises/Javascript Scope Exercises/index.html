  2. What is the difference between a method and function?
    A method is a function called inside an object. A function doesn't require an object.
  3. What does 'this' refer to when used in a Java method?
    this in Java refers to the object which that method belongs to.
  4. What does 'this' refer to when used in a JavaScript method?
    this in Javascript also refers to the object which that method belongs to. If there is no object, it refers to the window object.
  5. What does 'this' refer to when used in a JavaScript constructor function?
    this refers to the newly created object in the constructor function.
  6. Assume object x is the prototype for object y in Javascript. Object x has a method f( ) containing keyword 'this'. When f is called by x.f( ), what does 'this' refer to?

    when f is called by x.f, this refers to the x object.

    when f is called by y.f, this refers to the y object.

  7. What is a free variable in JavaScript?
    A free variable is a variable that is referred to by a function that is not one of its parameters or local variables.
