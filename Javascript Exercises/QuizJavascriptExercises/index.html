Javascript Exercises

  1. What is the difference between the following 2 statements? 
    setTimeout(booyah, 2000);
setTimeout(booyah(), 2000);
    The first one is passing booyah as a parameter to the setTimeout function. The second one is actually calling a function called booyah and whatever it returns is going to be passed as a parameter to the setTimeout function.
  2. What do the following 2 alerts display (answer without running the code)? 
    var myfunc = function(a, x) {
    return a * x;
};
var x = myfunc(2, 3);
var y = myfunc;
alert(x);
alert(y(2,3));
    The first alert shows the return value of x which is myfunc(2,3) and that is 6. The second alert shows the return value of the y function which is a variable function reference to myFunc, returning actually myFunc(2,3) which is 6.

  4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

    Unobtrusive Javascript is HTML with minimal Javascript inside.

    The practical application of it is a way to separate HTML from Logic using some DOM attachment event functions.

    One of the reasons for doing it is that you can keep HTML clean without Javascript mixed with it, effectively separating the website into three categories:

    1. content -> html
    2. presentation -> css
    3. behavior -> javascript
